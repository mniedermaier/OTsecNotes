\section{Overview}

Security testbeds provide controlled environments for learning and testing ICS security without risking production systems.

\posterinfo{
\textbf{Why use testbeds?}
\begin{itemize}
    \item Safe environment for attack practice
    \item No expensive hardware required (virtual)
    \item Hands-on protocol experience
    \item CTF and training scenarios
\end{itemize}
}

\section{CybICS}

\postersuccess{
\textbf{Most comprehensive protocol coverage}
\begin{itemize}
    \item 5 protocols: Modbus, OPC-UA, S7comm, DNP3, EtherNet/IP
    \item 13+ training modules with CTF challenges
    \item Docker or Raspberry Pi + STM32 deployment
\end{itemize}
\url{github.com/mniedermaier/CybICS}
}

\section{Labshock}

\posterwarning{Proprietary with limited free tier -- not fully open source.}

Fast setup OT security lab with built-in ELK Stack:
\begin{itemize}
    \item Real industrial protocols and telemetry
    \item Pentest Fury offensive module
    \item Pre-configured Kibana dashboards
    \item Min: 2 CPU, 2GB RAM, 10GB disk
\end{itemize}
\url{github.com/zakharb/labshock}

\section{GRFICS}

\posterinfo{
\textbf{3D visualization of attack impact}
\begin{itemize}
    \item Unity game engine graphics
    \item Chemical reactor simulation
    \item 5 VirtualBox VMs (v2) or simplified (v3)
    \item Includes OpenPLC, AdvancedHMI
\end{itemize}
}

\section{LICSTER}

\posterwarning{
\textbf{Physical hardware required (\texteuro500)}
\begin{itemize}
    \item Multiple Raspberry Pi boards
    \item Real I/O modules and timing
    \item Pre-built attack scenarios
    \item 3D-printed enclosure available
\end{itemize}
\url{github.com/thainnos/LICSTER}
}

\section{Deployment Types}

\posterinfo{
\textbf{Virtual} -- Docker/VMs, zero cost, quick setup\\
\textbf{Hybrid} -- Software + optional hardware\\
\textbf{Physical} -- Real equipment, realistic timing
}

\section{Key Takeaways}

\postersuccess{
\begin{itemize}
    \item Virtual testbeds are free and easy to deploy
    \item CybICS offers most protocol variety
    \item GRFICS excels at visualizing attack impact
    \item LICSTER provides best physical realism
    \item Most use MIT/GPL licenses (*Labshock: proprietary)
\end{itemize}
}
