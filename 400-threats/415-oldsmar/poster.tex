\section{\textcolor{accent}{\faIcon{info-circle}}\hspace{0.4em}Overview}

In February 2021, an attacker remotely accessed the Oldsmar, Florida water treatment plant and attempted to increase \textbf{sodium hydroxide (lye) from 100 ppm to 11,100 ppm} -- a potentially lethal concentration. An alert operator observed the intrusion in real-time and immediately reversed the changes.

\section{\textcolor{accent}{\faIcon{exclamation-circle}}\hspace{0.4em}Sodium Hydroxide Danger}

\posterwarning{
\textcolor{danger}{\faIcon{tint}}\hspace{0.2em}\textbf{NaOH at 11,100 ppm} is extremely dangerous. Normal treatment use is 25--100 ppm. At the attempted level: chemical burns, tissue damage, potentially fatal if ingested. However, multiple safety barriers existed.
}

\section{\textcolor{accent}{\faIcon{shield-alt}}\hspace{0.4em}Defense Recommendations}

\postersuccess{
\begin{enumerate}
    \item \textbf{Secure remote access} -- \textcolor{warning}{\faIcon{desktop}}\hspace{0.2em}No direct internet-to-SCADA
    \item \textbf{Implement MFA} -- Required for all remote access
    \item \textbf{Individual accounts} -- No shared passwords
    \item \textbf{Update OS} -- Replace Windows 7 and EOL systems
    \item \textbf{Segment networks} -- Separate IT, OT, internet
    \item \textbf{Monitor access} -- Log and alert on connections
\end{enumerate}
}

\postertip{
\textbf{Secure remote access:} Internet $\rightarrow$ VPN with MFA $\rightarrow$ Jump Server $\rightarrow$ OT Firewall $\rightarrow$ SCADA. Every step encrypted, logged, and monitored. Never install remote desktop tools directly on SCADA workstations.
}
