\section{\textcolor{accent}{\faIcon{info-circle}}\hspace{0.4em}Overview}

Attack vectors are the paths attackers use to access OT systems. OT environments face both IT-style attacks and \textbf{OT-specific threats} targeting industrial protocols and physical processes. Understanding vectors is essential for implementing effective defenses.

\posterinfo{
\textbf{Five vector categories:} Network-based (exploiting connectivity), Physical (direct equipment access), Supply chain (compromised vendors), Human (social engineering, insiders), and Wireless (RF communications).
}

\subsection{\textcolor{accent}{\faIcon{bug}}\hspace{0.3em}Protocol Exploitation}

\posterdanger{
\textbf{Industrial protocols lack authentication:} Modbus accepts commands from any source. DNP3 security rarely enabled. EtherNet/IP CIP has limited security. S7comm has weak auth. OPC Classic has known DCOM vulnerabilities.
}

\subsection{\textcolor{accent}{\faIcon{user-secret}}\hspace{0.3em}Man-in-the-Middle}

\begin{itemize}
    \item \textcolor{danger}{\faIcon{random}}\hspace{0.2em}ARP spoofing to intercept OT traffic
    \item \textcolor{danger}{\faIcon{pencil-alt}}\hspace{0.2em}Modifying commands between HMI and PLCs
    \item \textcolor{danger}{\faIcon{download}}\hspace{0.2em}Intercepting engineering uploads/downloads
\end{itemize}

\posterwarning{
\textbf{Remote access is the most exploited vector} in OT attacks. Ukraine grid attacks used stolen VPN credentials. Oldsmar used TeamViewer. Colonial Pipeline used a compromised VPN without MFA.
}

\postertip{
\textcolor{success}{\faIcon{shield-alt}}\hspace{0.2em}Remote access is the \textbf{most commonly exploited initial vector}. Implement MFA on all remote access as the single highest-impact control. No single defense stops all vectors -- layer controls across network, physical, supply chain, human, and wireless attack surfaces. Monitor the IT/OT boundary as the critical chokepoint.
}
