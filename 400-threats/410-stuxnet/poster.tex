\section{\textcolor{accent}{\faIcon{info-circle}}\hspace{0.4em}Overview}

Stuxnet (discovered 2010) is the \textbf{first true cyber weapon} -- malware designed to cause physical destruction of industrial equipment. It targeted Iran's Natanz uranium enrichment facility, destroying centrifuges while hiding its activities from operators.

\section{\textcolor{accent}{\faIcon{list-ul}}\hspace{0.4em}Key Facts}

\posterdanger{
Stuxnet demonstrated that \textbf{cyber attacks can cause physical destruction}. It fundamentally changed the threat landscape for critical infrastructure worldwide and lowered the barrier for future ICS attacks.
}

\section{\textcolor{accent}{\faIcon{bug}}\hspace{0.4em}Propagation Methods}

\begin{enumerate}
    \item \textcolor{danger}{\faIcon{usb}}\hspace{0.2em}\textbf{USB Drives} -- Exploited autorun and LNK vulnerabilities
    \item \textcolor{danger}{\faIcon{network-wired}}\hspace{0.2em}\textbf{Network Shares} -- Spread via Windows shares
    \item \textcolor{danger}{\faIcon{print}}\hspace{0.2em}\textbf{Print Spooler} -- Exploited MS10-061 vulnerability
    \item \textcolor{danger}{\faIcon{folder-open}}\hspace{0.2em}\textbf{Step 7 Projects} -- Infected shared project files
\end{enumerate}

\posterwarning{
Stuxnet \textbf{crossed the air gap} through infected USB drives carried by contractors and employees. Physical isolation alone is not sufficient protection against determined adversaries.
}

\section{\textcolor{accent}{\faIcon{search}}\hspace{0.4em}Target Identification}

Stuxnet was highly selective:

\begin{itemize}
    \item \textcolor{accent}{\faIcon{microchip}}\hspace{0.2em}Searched for Siemens Step 7 installations
    \item \textcolor{accent}{\faIcon{cogs}}\hspace{0.2em}Identified specific PLC configs (S7-315, S7-417)
    \item \textcolor{accent}{\faIcon{tachometer-alt}}\hspace{0.2em}Verified presence of specific frequency converters (Vacon, Fararo Paya)
    \item \textcolor{accent}{\faIcon{wave-square}}\hspace{0.2em}Checked drive frequencies 807--1210 Hz (centrifuge range)
    \item \textcolor{accent}{\faIcon{bullseye}}\hspace{0.2em}Only activated payload when \textbf{all conditions matched}
\end{itemize}

\section{\textcolor{accent}{\faIcon{shield-alt}}\hspace{0.4em}Defense Lessons}

\postersuccess{
\begin{enumerate}
    \item \textcolor{success}{\faIcon{plug}}\hspace{0.2em}\textbf{Air gaps aren't absolute} -- USB/contractor laptops bridge them
    \item \textcolor{success}{\faIcon{microchip}}\hspace{0.2em}\textbf{Monitor PLC integrity} -- Detect unauthorized logic changes
    \item \textcolor{success}{\faIcon{desktop}}\hspace{0.2em}\textbf{Verify operator displays} -- Cross-check with independent sensors
    \item \textcolor{success}{\faIcon{usb}}\hspace{0.2em}\textbf{Control removable media} -- Strict USB and media policies
    \item \textcolor{success}{\faIcon{project-diagram}}\hspace{0.2em}\textbf{Segment networks} -- Limit lateral movement in OT
    \item \textcolor{success}{\faIcon{band-aid}}\hspace{0.2em}\textbf{Patch systems} -- Apply updates where safely possible
\end{enumerate}
}

\postertip{
Stuxnet could have been detected through: monitoring for unauthorized Step 7 project modifications, anomalous PLC communication patterns, unexpected DLL injections in SCADA software, centrifuge performance deviations, and network traffic analysis for C2 communications. \textbf{Independent physical measurements} would have revealed the discrepancy between reported and actual centrifuge behavior.
}
