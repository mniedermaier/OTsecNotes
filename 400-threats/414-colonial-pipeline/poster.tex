\section{\textcolor{accent}{\faIcon{info-circle}}\hspace{0.4em}Overview}

The Colonial Pipeline ransomware attack (May 2021) demonstrated how \textbf{IT security incidents cascade into OT disruptions}. While the attack targeted IT systems only, the company shut down pipeline operations as a precaution, causing widespread fuel shortages across the US East Coast.

\section{\textcolor{accent}{\faIcon{key}}\hspace{0.4em}Initial Access}

\posterwarning{
\textbf{Attack vector:} A single compromised VPN account \textbf{without MFA}. Credentials found in a leaked password batch from another breach. The account was no longer actively used but remained enabled. One password, no MFA, no account cleanup.
}

\section{\textcolor{accent}{\faIcon{power-off}}\hspace{0.4em}Why OT Was Shut Down}

\posterinfo{
Colonial's \textbf{OT systems were not directly attacked}. The shutdown was precautionary:
\begin{itemize}
    \item \textcolor{warning}{\faIcon{dollar-sign}}\hspace{0.2em}Billing systems encrypted -- couldn't invoice
    \item \textcolor{warning}{\faIcon{exclamation-triangle}}\hspace{0.2em}Uncertainty about OT compromise extent
    \item \textcolor{danger}{\faIcon{bolt}}\hspace{0.2em}Risk through IT/OT connections
    \item \textcolor{info}{\faIcon{search}}\hspace{0.2em}Need for safe situation assessment
\end{itemize}
}

\section{\textcolor{accent}{\faIcon{shield-alt}}\hspace{0.4em}Defense Lessons}

\postersuccess{
\begin{enumerate}
    \item \textcolor{success}{\faIcon{fingerprint}}\hspace{0.2em}\textbf{Enforce MFA} -- On all remote access (VPN, RDP)
    \item \textcolor{success}{\faIcon{user-slash}}\hspace{0.2em}\textbf{Audit accounts} -- Disable unused accounts promptly
    \item \textcolor{success}{\faIcon{project-diagram}}\hspace{0.2em}\textbf{Segment IT/OT} -- Limit blast radius of IT compromise
    \item \textcolor{success}{\faIcon{dollar-sign}}\hspace{0.2em}\textbf{Plan for billing loss} -- OT ops may need IT systems
    \item \textcolor{success}{\faIcon{clipboard-list}}\hspace{0.2em}\textbf{IR planning} -- Know when and how to isolate OT
    \item \textcolor{success}{\faIcon{hdd}}\hspace{0.2em}\textbf{Offline backups} -- Resistant to ransomware encryption
\end{enumerate}
}

\postertip{
The most impactful single control: \textbf{MFA on all remote access}. Even with IT/OT segmentation, business processes may require both -- plan how OT can safely pause if IT is unavailable.
}
